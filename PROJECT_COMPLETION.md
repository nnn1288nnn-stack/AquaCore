## 🎓 使用教程

### 最简单的开始方式 (3 步)

```bash
# 1. 启动 Go API
cd golang-api && go run main.go

# 2. 在新终端启动 Python API (可选)
cd python-ai && python -m uvicorn main:app --reload

# 3. 运行测试脚本
bash test-api.sh
```

### 最常见的使用场景

1. **导航和数据提取**: 见 `examples.go` 示例 1
2. **表单自动填充**: 见 `examples.go` 示例 2
3. **多步工作流**: 见 `examples.go` 示例 8
4. **错误处理**: 见所有示例的 try-catch 块
